# 🌍 LapTheWorld

LapTheWorld is a full-stack web application designed to be the ultimate travel and race guide for Formula 1 fans attending Grands Prix around the world.

Born from firsthand experience at the Hungarian Grand Prix in Budapest, the platform solves a real problem — the lack of a single, comprehensive resource for race attendees.

Whether you're attending your first Grand Prix or your tenth, LapTheWorld brings everything you need into one place.


## 🚀 Features

### 🏁 Race & Championship
- 📅 **Race Calendar** — Full season calendar with country flags and live countdown timers
- 🏆 **Championship Standings** — Live driver and constructor standings updated after each race

### 🗺️ Travel & Circuits
- 🏎️ **Circuit Guide Pages** — Dedicated pages for each circuit including:
  - Track statistics
  - Travel info
  - Hotel recommendations
  - Local attractions
  - Weather
  - Packing guides

- 🗺️ **Interactive Travel Map**
  - Recommended hotels
  - Points of interest around each circuit

### 🎧 Smart Features
- 🎙️ **Voice Narration**
  - AI-generated audio using ElevenLabs API

- ⏱️ **Live Session Timing**
  - Real-time race weekend data powered by OpenF1 API

### 👥 Community
- 📝 **Community Blog**
  - Share travel tips
  - Circuit reviews
  - Personal race weekend experiences
