# LapTheWorld


LapTheWorld is a full-stack web application designed to be the ultimate travel and race guide for Formula 1 fans attending Grands Prix around the world. Born from a firsthand experience at the Hungarian Grand Prix in Budapest, the platform solves a real problem — the lack of a single, comprehensive resource for race attendees.
Whether you are attending your first Grand Prix or your tenth, LapTheWorld brings everything you need into one place.

Features:

🗓️ Race Calendar — Full season calendar with country flags and live countdown timers to each race weekend
🏆 Championship Standings — Live driver and constructor standings updated after every race
🏁 Circuit Guide Pages — Dedicated pages for each covered circuit with track statistics, travel info, hotel recommendations, local attractions, weather, and packing guides
🗺️ Interactive Travel Map — Map showing recommended hotels and points of interest around each circuit
🎙️ Voice Narration — AI-generated audio narration for each circuit powered by the ElevenLabs API
⏱️ Live Session Timing — Real-time race weekend session data powered by the OpenF1 API
📝 Community Blog — A space for fans to share travel tips, circuit reviews, and personal race weekend experiences
